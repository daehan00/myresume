# Resume Assistant

AI 기반 자기소개서 및 이력서 작성 보조 도구입니다. 채용 공고 분석부터 전략 수립, 초안 작성, 그리고 검토까지의 전 과정을 지원합니다.

### 바로가기
- **배포 주소:** [https://myresume-daehan00.streamlit.app](https://myresume-daehan00.streamlit.app)

### 주요 기능
- **공고 분석:** 채용 공고 및 사용자 경험 분석
- **전략 수립:** 기업 맞춤형 작성 전략 및 가이드라인 생성
- **AI 집필:** LLM을 활용한 자기소개서 초안 작성 및 심층 리뷰
- **멀티 모델 지원:** OpenAI, Gemini 선택 가능

### 🛠 기술 스택
- **Frontend:** Streamlit
- **AI Framework:** LangChain, LangGraph
- **Package Manager:** uv
- **Language:** Python 3.13
